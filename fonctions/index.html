Génère un tableau de nombres aléatoires.

tabAlea(int $nbrElement, int $valMin, int $valMax) : array<int, int>

Cette fonction crée un tableau d'entiers contenant un nombre donné d'éléments, chaque élément étant un entier aléatoire compris entre une valeur minimale et une valeur maximale.

Parameters
$nbrElement : int

Le nombre d'éléments à générer dans le tableau.

$valMin : int

La valeur minimale que peut prendre un élément.

$valMax : int

La valeur maximale que peut prendre un élément.

Tags
throws
InvalidArgumentException

Si le nombre d'éléments est < ou = à zéro, ou sphphpi $valMin est supérieur à $valMax.

Return values
array<int, int> —

Un tableau de nombres entiers aléatoires compris entre $valMin et $valMax.
